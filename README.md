# 📊 Projeto de Análise de Dados e Business Intelligence

📚 Educação Básica no Brasil

📌 Sobre o Projeto

Este projeto tem como objetivo analisar o impacto da desigualdade socioeconômica no desempenho escolar de alunos da educação básica no Brasil, utilizando técnicas de Análise de Dados e Business Intelligence (BI).
A proposta busca identificar padrões e relações entre fatores sociais e o rendimento acadêmico, apoiando a tomada de decisão baseada em dados.

🎯 Problema

A desigualdade socioeconômica influencia diretamente o acesso à educação de qualidade, refletindo em:

Desempenho escolar
Taxas de evasão
Acesso a recursos educacionais


🧠 Objetivo

Analisar como fatores como renda, acesso à internet e tipo de escola impactam a nota média dos alunos e a evasão escolar.

📂 Dataset

O dataset foi simulado com base em dados reais do cenário educacional brasileiro (INEP e outras referências), contendo 500 registros de alunos distribuídos pelas 5 regiões do Brasil.
Variáveis presentes:

Região e estado
Tipo de escola (Municipal, Estadual, Federal, Privada)
Renda familiar mensal
Acesso à internet (Sim/Não)
Escolaridade e anos de estudo do responsável
Distância até a escola (km)
Nota média (0 a 10)
Situação de evasão (Sim/Não)


📈 Metodologia

O projeto segue um pipeline automatizado em Python com as seguintes etapas:

Geração do dataset
Limpeza e tratamento dos dados
Análise exploratória (EDA)
Cálculo de correlações
Visualização dos dados
Exportação dos resultados


📊 Principais Análises

Correlação entre renda e desempenho (r = 0.55)
Impacto do acesso à internet nas notas
Comparação entre tipos de escola
Taxa de evasão por faixa de renda
Diferenças regionais no desempenho


📉 Visualizações
O projeto inclui os seguintes gráficos:

Gráfico de barras — nota média por região
Boxplot — distribuição de notas
Scatter plot — renda x nota
Barras empilhadas — tipo de escola
Comparação — com vs. sem internet
Heatmap de correlação


🖥️ Dashboard
A entrega final é um dashboard interativo em HTML, com navegação por abas e visualização dinâmica, sem necessidade de instalação.

Python

Matplotlib

⚠️ Observações
Os dados são simulados, portanto os resultados têm caráter educacional. A metodologia aplicada segue padrões reais de análise de dados.
